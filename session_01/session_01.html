Why learn machine learning?

Finance has become a data discipline.

  • Asset management, risk, trading, credit, fintech — all increasingly run on models fit to data.
  • Job postings that used to ask for “econometrics” now ask for machine learning.
  • You already know one such model: ordinary least squares. This course extends that instinct to more flexible tools.

The goal is not to make you a computer scientist. It is to let you read, build, and judge the models that finance now runs on.

What even is machine learning?

Your OLS regression is already a simple machine-learning model: it learns its coefficients from data. We build outward from there.

Variables: naming a value

A variable gives a value a name. The = sign means assign — “store this under that name” — not “equals”.

Once named, it stands for the value

Assignment can overwrite

Assigning to a name that already exists replaces its value. The old one is gone.

Can a name really appear on both sides of =?

Naming rules

Python requires:

  • letters, digits, and underscores only
  • a name may not start with a digit
  • no spaces: share price fails, share_price works
  • case matters: Price and price are two different names

Naming: your judgment

Both are legal. Only one will make sense to you next week:

x = 0.0212
weekly_return = 0.0212

Write names for the person who reads the code later. That person is usually you, a week from now.

8 · Errors are information

Part 8

An error message is a diagnosis, not a scolding.

An error is a diagnosis

Run this deliberately broken cell and read what comes back:

Nothing is damaged. Your other variables are fine. The runtime is fine.

Read it bottom-up

In Colab you get the full traceback. Read it from the bottom:

IndexError                          Traceback (most recent call last)
      1 prices = [183.56, 182.19, 179.87]
----> 2 prices[10]

IndexError: list index out of range
  • the last line names the error type and what went wrong
  • the arrow ----> marks the exact line that failed

Read the last line, look at the arrow, fix that line, run again. That resolves most errors before you search for anything.

The four errors of week one

error usually means first thing to check

The simple return

Yesterday’s close \(p_0\), today’s close \(p_1\). The simple return is

\[ r = \frac{p_1 - p_0}{p_0} \]

the percentage change in price — what one unit invested earned in a day. It is the basic object of this whole course.

Deliberately familiar: the same percentage change you know from econometrics growth rates.
