Running the notebooks locally

  1. Follow the setup guide once: Python, VS Code, the Python extension, and %pip install numpy pandas matplotlib scikit-learn.
  2. Put the notebook and a data/ folder in the same place, exactly as in the ZIP.
  3. In VS Code, File → Open Folder and choose that folder. Not the file: the folder.
  4. Open the notebook, pick your Python when asked, and run the first cell.
If you skip the data files entirely, the notebooks still work: when they cannot find a local data/ folder they download what they need from this repository. You only need the CSVs if you want to work with no internet at all.

A note on the lectures

Each rendered lecture is an HTML file plus a folder of assets next to it (session_03.html and session_03_files/). Downloading the HTML on its own gives you a broken page, so take the ZIP if you want the slides offline. The live code cells also need the page to be served over HTTP rather than opened from a file:// path, which is a browser restriction, not a course one. Online, the links on the sessions page always work.
