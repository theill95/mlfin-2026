Working locally, from Session 3

Colab keeps working for everything on this page, so nothing here is urgent. But the exam is written on your own laptop in VS Code with no internet, so it is worth setting this up early and getting used to it. Do the four steps in order; each link is the official guide.

  1. Install Python. Download the latest release for your system and, on Windows, tick Add python.exe to PATH in the installer. python.org/downloads
  2. Install VS Code. The editor itself. code.visualstudio.com/download
  3. Add the Python extension and run your first file. This walkthrough covers the extension, selecting an interpreter, and running code. VS Code Python tutorial · notebooks in VS Code: Jupyter in VS Code
  4. Install the packages we use. Open a new notebook in VS Code and run %pip install numpy pandas matplotlib scikit-learn in a cell. (If you would rather use a terminal, pip install with the same names does exactly the same thing.) Background on what pip does: Installing packages

Session 3 opens by walking through this, so come with it installed if you can. If a step fails, bring the error message and we will look at it together.
