Machine Learning in Finance

Programming and machine learning for finance and economics students, starting from zero. Everything you need for each session is below.

You appear to be viewing this file from your computer. The lecture's live code and the “Open in Colab” buttons work best from the online version your instructor links on the learning platform.

Start here

If this is your first visit, two minutes on this card will save you time later. Everything on this site is free to use, and nothing needs to be installed to begin.

What each session gives you

  1. The interactive lecture. The slides from class, with the code still runnable. Use it in class if you like, and again afterwards when you want to try changing something and see what happens.
  2. The exercises. Short tasks that drill one idea at a time, rated one to five stars. You are not expected to finish them all. Hints and full solutions are folded under each task, so you are never stuck for long.
  3. The case. One investigation of real market data that continues across all four sessions. Part 2 picks up the numbers Part 1 worked out, and so on. It is where the separate pieces become an analysis.

Do the exercises before the case. The exercises teach the moves, the case asks you to choose which move to make.

Using the interactive lecture

It runs entirely in your browser, so there is nothing to install and nothing to download. Code cells are editable: change a number, press Run, and see what happens. Sessions 3 and 4 load NumPy, pandas and matplotlib into the page, so give the first cell a few seconds.

Three keys worth knowing

  • m opens a menu listing every slide by title, so you can jump straight to the part you want.
  • e lays the whole lecture out as one long page you can scroll and search with your browser's own find. This is the one to use when you are revising. Press e again to go back to slides.
  • o zooms out to show all the slides at once.

Working on the notebooks

The exercises and the case open in Google Colab, which runs Python for you in the browser. Save your own copy before you start (File, then Save a copy in Drive), or your work is lost when you close the tab. If you would rather work on your own machine, the setup guide walks through it and the downloads page has every notebook and data file. Both routes use exactly the same data, so your numbers will match the ones shown in class.

When you get stuck. The cheatsheet lists everything taught so far, with a worked example of each and the session it came from. The resources page covers the official documentation, free books, and where AI genuinely helps rather than hurts. If you are still stuck, ask a friend or email jobo@econ.au.dk.

How to use these materials

  • Lecture: click Interactive lecture. Arrow keys move between slides; the code cells run in your browser, so you can edit them and press Run.
  • Exercises & case: click a button to open the notebook in Colab, then File → Save a copy in Drive so your work is your own. Hints and solutions are folded inside each task, and the data loads by itself with nothing to upload.
  • Sessions 1 and 2 need nothing installed. A browser and a Google account are enough. From Session 3 you can also work locally in VS Code, which is what you will use at the exam: the setup guide walks through it.
  • Cannot remember a function? The cheatsheet lists everything the course has used, each with a worked example.
