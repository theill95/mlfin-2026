Resources

The course materials are complete on their own: the lectures, the exercises, the case and the cheatsheet are all you need. Everything here is either the official reference you should learn to use, or something extra for when you want to go further.

needed use it during the course  ·  extra only if you want more

Official documentation needed

Learn to read these rather than searching for a blog post. They are written by the people who wrote the code, they match the version you have, and the reference part of them is already on your machine: hovering a name in VS Code shows the same text.

  • docs.python.org  the language and its standard library. The tutorial is a good second pass over Sessions 1 and 2.
  • numpy.org/doc  arrays and linear algebra. Start with NumPy for absolute beginners.
  • pandas.pydata.org/docs  tables. The user guide explains the ideas; the API reference is what you look details up in.
  • The matplotlib gallery  a wall of example charts, each with the code that made it. Find the picture closest to what you want and read how it was done. This is how everybody learns plotting, including people who do it for a living.

Taking the documentation offline needed

Two things are worth knowing, because there are times you will not have a search box.

The reference is already on your machine. Every package carries its own manual inside it. Hover a name in VS Code, or call help(name) in a cell, and you get the signature and the parameter list. That is the same text the reference pages on the website are generated from.

The full sites can be downloaded. Python publishes its whole documentation as a single archive at docs.python.org/3/download.html, and the NumPy and pandas sites offer offline copies of their versioned documentation too. Worth doing once, in good time, rather than on a day you need it.

Practising with the wifi switched off is a genuinely useful exercise. It is surprisingly quick to get used to help() and hover, and much easier to learn now than under time pressure.

Books, free and legal needed

  • An Introduction to Statistical Learning, with Applications in Python (James, Witten, Hastie, Tibshirani, Taylor). Free PDF from the authors. This is the conceptual reference for the machine-learning half of the course, and Session 4 follows its opening chapter closely. If you read one thing beyond the slides, read this.
  • Python for Data Analysis (Wes McKinney, who wrote pandas). Free to read online. The chapters on NumPy and pandas are the natural next step after Session 3. extra

More practice extra

Only if you have finished the exercises and want more. None of this is examined.

  • Kaggle Learn  short free micro-courses, each a few hours, that run in your browser with nothing to install. The Python, Pandas, Data Visualization and Intro to Machine Learning ones overlap this course closely and are a good second explanation if something has not clicked.
  • Kaggle datasets  thousands of real datasets to practise on. Pick something you are actually curious about: you will learn more from a question you care about than from another tutorial.
  • Exercism, Python track  small programming puzzles with feedback from volunteer mentors. Good for making the language itself feel automatic.

The data this course uses extra

Daily closing prices for eleven US instruments (AAPL, MSFT, NVDA, JPM, KO, PG, XOM, JNJ, WMT, DIS, and the S&P 500 ETF SPY) from 2015 to 2024, split- and dividend-adjusted. They were downloaded once from Yahoo Finance and committed to the repository, so the materials never depend on a live connection and everyone works with exactly the same numbers.

The script that built them is tools/build_dataset.py in the repository, if you want to see where the files came from or rebuild them for other tickers. The files themselves are on the downloads page.

Using AI on this course needed

You may use it, and you should learn to. The distinction that matters is between using it to understand and using it to avoid understanding.

  • Good: "explain what this error means", "why does pandas want .copy() here", "give me a hint, not the answer", "what is this function's second argument called".
  • Bad: pasting an exercise and copying what comes back. It reads as though you learned something, right up until the exam, where there is no internet and no assistant.

A fair test: could you now write it again, from a blank cell, without help? If not, you have not finished the exercise.
